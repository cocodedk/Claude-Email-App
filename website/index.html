Features

Built for command-and-reply work.

Six deliberate capabilities. Nothing decorative, nothing enterprise — the surface area a person working with agents actually needs.

01 · IMAP + SMTP

Real email transport.

Standards under the hood. No proprietary lock-in, no bespoke protocol, no account you can’t leave.

02 · Android Keystore

Credentials sealed to the device.

Passwords are encrypted by hardware-backed keys. They never sync and never leave the phone.

03 · Command & reply

Conversations, not threads.

Agents and tasks are first-class. Title, agent badge, state — readable in one glance.

04 · Shared-secret auth

Authorize without passwords.

Route commands to the claude-email service with a shared secret. Agents stay scoped and revocable.

05 · Offline queue

Send now, deliver when ready.

Commands compose offline and queue locally. They fire the moment the network returns.

06 · Dark-first UI

Material 3, tuned for focus.

Calmer than Slack, more readable than Outlook. Built for long reading sessions, not consumer chat.

Install

Three steps to first command.

No store listing. No account to create. You hold the keys — the app just uses them.

  1. 01 / Download

    Grab the signed APK.

    Pull the latest release straight from GitHub. Reproducible, signed, and checksummed.

  2. 02 / Install

    Install on Android 7.0+.

    Allow “install unknown apps” for your file manager once. Android handles the rest.

  3. 03 / Configure

    Enter IMAP/SMTP credentials.

    Credentials are encrypted locally by the Keystore. They never leave the device.
